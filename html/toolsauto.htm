Auto Scheduler

This command can be accessed from the Tools Scheduler

Use this command to create a schedule before the start of any season.

This command is inactive during the course of a season.

Creating a schedule

Select 2,4,6 or 8 games between all teams in the same division

Select 2,4,6 or 8 games between all teams in the same conference

Select 2,4,6 or 8 games against all teams in the other conference

(games selected will be split even between home games and away games)

Select up to 98 games to be played by each team in the league

If the total games created by the first 3 options above is greater than the total games to be played, the schedule will automatically be increased to a higher total games played.

if the total games created by the first 3 options above is less than the total games to be played, the rest of the schedule will filled with randomly selected games versus conference opponents.

Click the Create Button to have the computer automatically create a schedule

If the computer is unable to create a schedule you will be notified otherwise you will be returned to the League Schedule Screen where you can elect to edit, cancel or save the schedule.
