Franchise New

Use this command to setup or edit the current league and its format or to create your own custom leagues.

Select a name for your league

note: leagues starting with the name "default" can only be used to create new leagues and can not be used for your franchises.

Before saving your league (Click Create) select from the following options

draft league Hold a fantasy draft with all the players in the current league.

Existing Players- Saved league will use the same players on the same teams as the current league

Fictional Draft- Hold a fantasy draft using only randomly generated fictional players

To change a team name of an existing team

- select the team in the Current Teams box.

- type the new name (16 characters maximum) in the Edit Team Name box.

- click Save Changes

Note: Team Names can not be changed for current leagues.

We recommend opening and using default leagues to create your own custom leagues.

Select a team to add to the league from the Select Teams box then click the Add to League Button. Repeat for every team to include in your league.

Select a team from the Current box and use the Delete from League to remove a team from the league.

Assign a team to a division by clicking on a team from the Current box and then selecting a blank slot under the Division you want to assign that team to.

Remove a team from a division by selecting a team and then double clicking.

Select Injuries by selecting the appropriate button.

None - no injuries occur

Random - injuries occur at about the same rate for all players.

Stats Based - Injuries are based on the actual player stats.

Career Play

checking will activate the career menu commands as necessary.

computer teams will make general managerial decisions based on career projections as well as current performance.

Unchecking will disable the career menu items.

computer teams will make general managerial decisions based only on current performance.

Import Arena/City File

Click to import an existing arena file or just edit if you already know the name

If the arena file named here does not exist a random arena file will be created.

You can edit the file by selecting the Franchise-Edit Arena/City File Command

To toggle from player control to computer control, select a team from the Current box and then click the Control box.

Player control teams automatically default to the following settings

Save Play by Play - on

Save Box Scores - on

Receive Trade Offers - on

To change the options use the Options Button

These options can be changed any time during a season.

With the exception of career play, injuries, trading and player control, any other changes must be made before the current league has started league play. If the season is underway you must use the League Schedule command Reset Button to start the season over.
