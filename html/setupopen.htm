Setup Load

Use this command to open an existing league

After opening a league file starting with the name "default" you can use  Setup New League to setup your own custom leagues as they can not be used for franchises.

The following options allow you to specify which file to open:

File Name

Type or select the league filename you want to open. This box lists files with the extension

.lge in the List Files of Type box. If you type a league filename that doesn't exist a new league

file will be created.

List Files of Type

Select the type of file you want to open:

.lge

Drives

Select the drive in which Jump Shot stores the league file that you want to open.

Directories

Select the directory in which Jump Shot stores the league file that you want to open.
